# Films de Ketard

Projet React + Vite compatible Vercel/GitHub.

## Structure
- `index.html` pointe vers `/src/main.jsx`
- `src/main.jsx` monte l’application React
- `src/App.jsx` contient le code complet actuel

## Vercel
- Framework Preset : Vite
- Build Command : `npm run build`
- Output Directory : `dist`

## Variable d’environnement recommandée
`VITE_TMDB_API_KEY=ta_cle_tmdb`
